autoCollapse:Boolean

Determines if the reset() method is called before setting the selected item. This will allow multiple content areas to be open at the same time. Set this flag to false if only one content area should only ever display.

arrange()method 
public function arrange():void

This method arranges all the items based on their height property. It is called after the selected item's flag was set which will mean any reference to the item's height property will produce the header + content height. If the selected flag is false the property will return the header height only.
